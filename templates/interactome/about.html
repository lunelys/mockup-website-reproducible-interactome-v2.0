This interface enables to search experimental evidences for different species interactions. These experimental evidences have been curated from both explicit and implicit redundancies, as described in Melkonian et al.


The reproducible-interactome v2.0 was coded by Lunelys Runeshaw during a 2 months first year of a master's degree internship, with Gwenael Rabut as supervisor.


This web interface has been developed by: Lunelys Runeshaw. This is the mockup version of the website, with only a few NAM7 interactions in the database, to show the functionalities.



All tools used are available on GitHub.
